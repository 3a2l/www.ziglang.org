Releases

Releases are provided in source and binary form. We currently recommend the master binaries, as Zig is a fast moving target and there are many improvements and fixes since 0.2.0.

You can also install Zig from a package manager, such as homebrew for MacOS.

Please consider donating $1/month to support the project.
